# ⚛️ Projetos Base em React

Este repositório foi elaborado pelos alunos do curso de **Desenvolvimento de Software Multiplataforma da Fatec Itapira**. 

O objetivo deste espaço é centralizar pequenos projetos base, templates e exercícios práticos em **React** para apoiar nossos estudos e solidificar conceitos essenciais do desenvolvimento web e multiplataforma.
